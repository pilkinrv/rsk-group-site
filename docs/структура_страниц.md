# Структура страниц сайта РСК ГРУПП

> **Навигация:** ориентир — [Световые технологии](https://www.ltcompany.com/catalog/by-product-type): каталог по типам продукции, сервисы, портфолио.

> **Изменение:** раздел МАФ исключён — компания больше не занимается малыми архитектурными формами.

## Структура меню (по логике ltcompany.com)

| Раздел | Подразделы |
|--------|------------|
| **Каталог** | Продукция (наружное/внутреннее), Решения, Портфолио |
| **Сервисы** | Материалы для скачивания (BIM, IES), Регистрация проектировщиков |
| **Информация** | О компании, Новости, Контакты |

## Описание разделов

| Раздел | URL | Назначение |
|--------|-----|------------|
| Главная | / | Hero, блок «ЗАКАЗЧИКУ», переход к разделу проектировщиков |
| ЗАКАЗЧИКУ | /zakazchiku/ | Производство полного цикла, проекты, технологии, завод. Только контактная форма |
| Проектировщикам | /proektirovshchikam/ | Каталог (~200 позиций), регистрация, чат в Telegram |
| О компании | /o-kompanii/ | О нас, история, производство, ценности, экскурсия, сервис, отзывы, полезное |
| Новости | /novosti/ | Лента новостей |
| Портфолио | /portfolio/ | Кейсы: парки, ЖК, общественные пространства |
| Доставка и оплата | /dostavka-i-oplata/ | Условия доставки и оплаты |
| Гарантии | /garantii-ot-proizvoditelya/ | Гарантия 5 лет, условия |
| Контакты | /kontakty/ | Адрес, телефон, форма, карта |

## Карточка товара (раздел проектировщиков)

- Описание как **решение** (куда и зачем применять)
- Характеристики
- 3D-модель (интерактивная)
- Габаритный чертёж
- BIM-модель (Revit .rfa, др.) — скачивание после регистрации
- Рендер
- IES-файл — скачивание после регистрации
- Ссылки на референс-объекты
- Кнопка «Получить консультацию» / «Заказать»

## Регистрация проектировщика

- Поля: ФИО, компания, должность, email, телефон
- Подтверждение email обязательно
- Доступ после регистрации: BIM/IES, чат в Telegram
