# Проект: Разработка сайта РСК ГРУПП

> Новый корпоративный сайт производителя светодиодного осветительного оборудования.

## О проекте

- **Клиент:** ООО «РСК ГРУПП»
- **Текущий сайт:** https://рскгрупп.рф
- **Тип:** Статические страницы + каталог продукции с 3D-моделями
- **Статус:** В разработке
- **Дата создания:** 2026-03-07

### Важно

**МАФ (малые архитектурные формы) исключены** — компания больше не занимается этим направлением. Фокус: светодиодное освещение (наружное и внутреннее).

## Стек технологий

| Компонент | Технология |
|-----------|------------|
| Фреймворк | Next.js 14+ (App Router) |
| Язык | TypeScript |
| UI-компоненты | **shadcn/ui** (Radix + Tailwind) |
| 3D | React Three Fiber + Three.js |
| Стили | Tailwind CSS |
| Формат 3D | GLB/GLTF |

## Особенности

- **Статические страницы** — главная, о компании, доставка, гарантии, объекты, контакты
- **Каталог продукции** — наружное и внутреннее освещение
- **3D-модели** — интерактивная анимация продукции на карточках товаров
- **AI SEO** — оптимизация под поиск ИИ-моделями (JSON-LD, семантика, структурированный контент)

### Запуск сайта

```bash
cd website
npm run dev
```

### Добавление компонентов shadcn/ui

```bash
cd website
npx shadcn@latest add <компонент>
# Примеры: button, card, input, dialog, badge
```
